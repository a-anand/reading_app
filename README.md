# Reading Practice App for Kids

A simple, colorful web application to help 7-year-olds practice reading words.

## Features

- **100 Age-Appropriate Words**: All words are 4+ letters and suitable for 7-year-olds
- **Large, Easy-to-Read Display**: Words shown in large, colorful text at center of screen
- **Pronunciation Guide**: Shows phonetic spelling below each word
- **Text-to-Speech**: "Read Aloud" button speaks the word clearly
- **Easy Navigation**: Previous/Next buttons to move through words
- **Keyboard Shortcuts**:
  - Left Arrow: Previous word
  - Right Arrow: Next word
  - Spacebar: Read aloud
- **Progress Counter**: Shows current word number (e.g., "Word 1 of 100")
- **Kid-Friendly Design**: Colorful gradient backgrounds and large buttons

## How to Use

1. **Open the app**: Simply double-click `index.html` in your browser
2. **Read the word**: The word is displayed in large letters at the center
3. **Check pronunciation**: The phonetic spelling is shown below the word
4. **Listen**: Click the "ðŸ”Š Read Aloud" button to hear the word
5. **Navigate**: Use Previous/Next buttons or arrow keys to move through words

## Technical Details

- Pure HTML/CSS/JavaScript (no dependencies needed)
- Uses Web Speech API for text-to-speech
- Works in all modern browsers (Chrome, Firefox, Safari, Edge)
- Responsive design works on tablets and phones
- Speech is slowed down slightly (0.8x speed) for easier learning

## Browser Compatibility

Text-to-speech works best in:
- Google Chrome
- Microsoft Edge
- Safari (on macOS and iOS)

## Customization

You can easily modify:
- The word list (edit the `words` array in the JavaScript)
- Colors and styling (edit the `<style>` section)
- Speech rate and pitch (edit the `speakWord()` function)

## Tips for Parents

- Practice a few words each day
- Encourage your child to repeat the word after hearing it
- Ask them to use the word in a sentence
- Revisit difficult words by navigating back to them
